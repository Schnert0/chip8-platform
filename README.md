# CHIP-8 Platform
A CHIP-8 Abstraction Layer for porting CHIP-8 games to C.

### NOTE: This project is incomplete and only contains a subset of the CHIP-8's functionality

## Requirements
```
SDL2
```

## Compiling
Run the bash script to compile all programs:
```
bash make.sh
```
